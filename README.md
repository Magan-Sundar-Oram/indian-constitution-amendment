# AmendMaster

Mobile-first Constitutional Amendments of India learning and quiz app.

Open `index.html` in a browser to use it offline. The app includes:

- All 106 Constitutional Amendment Acts from 1951 to 2023.
- Separate data source at `data/amendments.json`.
- 1,060 generated exam-style questions in `data/questions.json`.
- Practice, exam, amendment-wise, random, weak-area, revision, and daily modes.
- Dashboard, search, topic filter, bookmarks, explanations, progress, XP, badges, streaks, and local leaderboard.
- Light and dark mode.

Progress is stored locally in the browser with `localStorage`; no backend is required.
